# Schronisko dla zwierząt – Python i C#

Projekt przedstawia prostą aplikację konsolową symulującą działanie
schroniska dla zwierząt. Program został wykonany w językach **Python**
oraz **C#**
---

## Wykorzystane klasy

### Animal
Reprezentuje zwierzę w schronisku.
- imię
- gatunek
- status adopcji

### Volunteer
Reprezentuje wolontariusza opiekującego się zwierzętami.
- imię

### Adoption
Reprezentuje proces adopcji.
- przypisuje zwierzę do wolontariusza

---

## Funkcjonalność programu

Po uruchomieniu aplikacji użytkownik może:
1. dodać nowe zwierzę do schroniska
2. dodać wolontariusza
3. przeprowadzić adopcję zwierzęcia
4. wyświetlić listę zwierząt wraz ze statusem adopcji
